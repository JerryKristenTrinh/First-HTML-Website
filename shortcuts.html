Example Usage

Input Emmet code:

(h$.value-$.#Question-$.{Question $}+br)*4

Output HTML code:


<h1 class="value-1" id="Question-1">Question 1</h1><br>
<h2 class="value-2" id="Question-2">Question 2</h2><br>
<h3 class="value-3" id="Question-3">Question 3</h3><br>
<h4 class="value-4" id="Question-4">Question 4</h4><br>

Rendered Website Output:

Question 1


Question 2


Question 3


Question 4


Emmet Example Explanation

h = Header Tag (e.g., h1, h2, h3)

. = Class Selector

$ = Counter / Loop number (auto-increments)

# = ID Selector

{str} = Content (text inside the tag)

+ = Sibling Operator (creates an element at the same level)

*N = Multiplication (repeats the preceding element or group N times)

() = Grouping (used for complex nested structures and applying multiplication to a group)

Emmet Abbreviation Cheatsheet

Basic Elements & Attributes

div or **(no tag)** = Creates a `<div>` tag (implicit tag)

p = Creates a `<p>` tag

a = Creates an `<a>` tag

img = Creates an `<img>` tag with `src` and `alt` attributes

input = Creates an `<input>` tag with `type="text"`

input:email = Creates `<input type="email">` (various input types available)

meta:utf = Creates `<meta charset="UTF-8">`

link:css = Creates `<link rel="stylesheet" href="style.css">`

script:src = Creates `<script src="main.js"></script>`

doc or html:5 = Generates full HTML5 boilerplate

Operators & Modifiers

> (Child) = Nest elements inside each other.
Example: div>ul>li expands to `<div><ul><li></li></ul></div>`

+ (Sibling) = Place elements next to each other at the same level.
Example: div+p+img expands to `<div></div><p></p><img>`

^ (Climb-up) = Move up one level in the tree.
Example: div>ul>li^p expands to `<div><ul><li></li></ul><p></p></div>`

*N (Multiplication) = Repeat an element or group N times.
Example: ul>li*3 expands to `<ul><li></li><li></li><li></li></ul>`

$ (Numbering) = Place cursor/number counter.
Example: h${Item $}*3 expands to `<h1>Item 1</h1><h2>Item 2</h2><h3>Item 3</h3>`

$@- (Reverse Numbering) = Decrement counter.
Example: li$@- {Item $}*3 expands to `<li>Item 3</li><li>Item 2</li><li>Item 1</li>`

$@N (Start Numbering at N) = Start counter from a specific number.
Example: li$@3 {Item $}*3 expands to `<li>Item 3</li><li>Item 4</li><li>Item 5</li>`

.class (Class) = Add a class attribute.
Example: p.my-paragraph expands to `<p class="my-paragraph"></p>`

#id (ID) = Add an ID attribute.
Example: div#main-content expands to `<div id="main-content"></div>`

[attr="value"] (Attributes) = Add custom attributes.
Example: a[href="#top" title="Back to top"] expands to `<a href="#top" title="Back to top"></a>`

{text} (Text) = Add text content inside the tag.
Example: p{Hello World} expands to `<p>Hello World</p>`

{} (Empty Content) = Ensure the tag has no content (useful for self-closing tags, though not strictly needed).
Example: img{} expands to `<img>`

() (Grouping) = Group complex expressions together.
Example: (header>nav)+main+(footer>p) expands to: 


<header>
    <nav></nav>
</header>
<main></main>
<footer>
    <p></p>
</footer>

Dummy Text (Lorem Ipsum)

lorem = Generates a 30-word Lorem Ipsum paragraph.
Example: p>lorem expands to a `<p>` with dummy text.

loremN = Generates N words of Lorem Ipsum.
Example: p>lorem10 generates 10 words.
